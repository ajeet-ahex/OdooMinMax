Overview

This Module is designed to set the Minimum order quantity of any products in ecommerce website. You can set minimum order quantity of any product without any hussle and without any code level changes. Install this plugin in your project and follow these basic steps to achieve the desired outcomes.

Features

After Installing, you will get the Minimum Order Quantity Field to add the minimum Amount to set for that particular Product

Once Done, The work of plugin will start and you don't need to do anything after that. This is end User Screen
